SO Classification Tool Documentation

The Storm Overflow Classification Tool is designed to assist in the evaluation and classification of storm overflows based on specific hydraulic criteria. This tool focuses on comparing the consent pass forward flow, Formula A, and actual model predictions to determine the classification status of storm overflows as either satisfactory, substandard, or unsatisfactory. This documentation provides guidance on how to use the tool effectively.
