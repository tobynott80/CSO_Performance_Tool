Getting Started


Step 1: Create a New Location


  • Navigate to the Dashboard - You will see the "Location Tests" tab.
  • Add New Location - Click on the "New" button. You will be prompted to enter details for the new location.
  • Enter Location Details - Provide a name for the location and any other relevant details as required.
  • Save the Location - After entering all necessary information, click "Save" to create the new location.
    •

    Step 2: Create a Run

    Once you have defined a location, the next step is to create a run. A run is a specific assessment session that allows you to evaluate storm overflows within the selected location using various tests.

  • Access the Runs Section: After clicking a location you'll be in the "Runs" tab.
  • Initiate a New Run: Click on the "New" button.
  • Provide Run Details: Give your run a descriptive name and optionally add a description to detail the purpose or focus of this particular run.


    • Step 3: Pick Tests and Set Parameters

    With a run created, you can now choose which tests to include in your assessment.

  • Select Tests: You'll see a list of available tests, each with a checkbox next to it. Check the boxes for the tests you want to include in your run.
  • Configure Test Parameters: Some tests require you to set specific parameters. Click on a test to expand its settings, where you can input the necessary parameters and add a description for clarity.
  • Finalize Tests: Once you've selected your tests and configured the required parameters, save your selections to include them in the run.


    • Step 4: Execute the Run and View Results

    After configuration and pressing "Create", the tool will execute the run and begin running the tests.

  • Start the Run: Click the "Start Run" button to initiate the testing process. The tool will now perform the selected tests based on the provided parameters.
  • Monitor Progress: You can view the status of the run in real-time. Once all tests are completed, the run status will change to "Complete"


    • Step 5: Analyze the Results

    Upon completion of the run, it's time to review and analyze the results.

  • Access the Results Table: Navigate to the completed run and click on it to view a summary table of the results.
  • Detailed View: For more detailed information on a specific test result, click on the corresponding row in the table.
  • Visualizations: The tool provides visualization options for better understanding the data. Access these by selecting the "Visualizations" tab within the run.
  • Download and Export: You have the option to download or export the results for further analysis or reporting.
