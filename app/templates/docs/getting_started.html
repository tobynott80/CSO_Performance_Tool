Getting Started


Step 1: Create a New Location


  • Navigate to the Dashboard - You will see the "Location" table.
  • Add New Location - Click on the "New" button..
  • Enter Location Details - Provide a name for the location.
  • Save the Location - Click "Save" to create the new location.
    •

    Step 2: Create a Run

    Once you have defined a location, the next step is to create a run. A run is where you will choose which tests to perform on your data.

  • Access the Runs Section: After clicking a location you will see the "Runs" table.
  • Initiate a New Run: Click on the "New" button.
  • Provide Run Details: Give your run a name and optionally add a description to detail the purpose or focus of this particular run.
  • Choose Tests: Pick which test you will run by selecting the checkboxes. You have the choice between 1. Dry Dry Day Discharges 2. Rainfall Discharges 3. Formula A and Consent


    • Step 3: Upload files and Set Parameters

    With your tests now chosen, its time to choose the input files and set the parameters.

  • Upload Files: Based on which tests you've chosen, you'll then see which files are required to be uploaded.
  • Configure Test Parameters: This will only be required for test 3.
  • Validate: Once all the tests and parameters are uploaded, press the validate button to check the files are correctly formatted.
  • Validation Failed?: If you've ran in to an error please check the error message and act accordingly. The majority of the errors will be down to incorrect format of the input files but it could also potentially require you to restart the application and rerun ./setup.bat if all else fails.


    • Step 4: Create a Run or choose assets if required

    If the validation has passed you will then see either 'Create Run' or have the option to select assets.

  • No Extra Assets: If this is the case you can simply create a run and it will take you to the results page.
  • Extra Assets: If the application does detect multiple assets then please tick the checkboxes of the assets you'd like to run and then create the run.


    • Step 5: Analyze the Results

    Upon completion of the run, it's time to review and analyze the results.

  • Access the Results Table: Navigate to the completed run and click on it to view a summary table of the results.
  • Detailed View: For more detailed information on a specific test result, click on the specific result you would like to see.
  • Visualizations: The tool provides visualization options for better understanding the data. Access these by selecting the "Visualizations" button within the run.
  • Download and Export: You have the option to download or export the results for further analysis or reporting.
