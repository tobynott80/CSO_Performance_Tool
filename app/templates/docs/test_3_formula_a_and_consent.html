Test 3 - Formula A and Consent

What is Consent Pass Forward Flow?

  • This is the flow the storm overflow is required to pass forward onto treatment before it is allowed to spill. It effectively controls how dilute the spill flow is.

What is Formula A:

  • Formula A is calculated from various model parameters (population, trade flow, per capita consumption, baseflow infiltration).

How does the application calculate the results of the Formula A and Consent Test?

  • If the value is lower than the consent FPF, then the asset is ‘Unsatisfactory’.
  • If the value is only lower than Formula A, then the asset is ‘Substandard’.
  • If the value is greater than both, then it is ‘Satisfactory’.

How does the application calculate individual results?

Formula A Compliance:

  • Unsatisfactory: If either the peak or average initial performance falls below formula A.
  • Satisfactory: If both the peak and average initial performance meet or exceed formula A.
  • None: If formula A is not provided, implying no specific standard to check against.

Consent FPF Compliance:

  • Unsatisfactory: If either the peak or average initial performance falls below consent FPF.
  • Satisfactory: If both the peak and average initial performance meet or exceed consent FPF.
  • None: If consent FPF is not provided, implying no specific consent threshold to evaluate against.
