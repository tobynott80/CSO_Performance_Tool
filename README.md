# Arup Water Quality

## Usage

1. Clone/Download this repo

   - `git clone https://github.com/tobynott80/FlaskTest.git`
   - [Alternative zip download](https://github.com/tobynott80/FlaskTest/archive/refs/heads/main.zip)

2. Install Flask
   `pip install Flask`
3. Run the app
   `python .\run.py`
